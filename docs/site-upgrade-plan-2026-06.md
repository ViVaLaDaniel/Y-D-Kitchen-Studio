## Technical notes

The repo is intentionally small: React + Vite with content in `src/data.ts`, composition in `src/App.tsx` and styling in `src/styles.css`. This is enough for the current landing page. If SEO pages grow beyond a few sections, move to a route-capable setup or static page generation rather than cramming every keyword into one page.
